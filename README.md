# FullAutoZombieBuster
## About
Twitter(現X)でAIによる自動返信を行っているBotアカウントを自動的に処理（ブロック/ミュート）するユーザースクリプトです。 

一般条件として、以下の条件に全て一致した場合にBotと判定されます。

- 1 表示名に日本語が含まれていない
- 2 認証済みアカウントである
- 3 表示名かプロフィールに特定のキーワードが含まれている
- 4 プロフィールに日本語が含まれていない

また、特殊条件として、  
- A フォロー中のユーザーは処理しない(最優先)  
- B 認証済みで表示名とプロフィールに日本語が含まれていない場合、他条件によらず処理 (簡体字が含まれている場合は即座に処理)  
- C 認証済みでプロフィールが空欄の場合、他条件によらず処理  

があります。誤爆の可能性があるため、BとCはデフォルトで無効になっています。

## Install
Tampermonkeyなどを通じてインストールしてください。
