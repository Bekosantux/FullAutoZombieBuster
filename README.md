# FullAutoZombieBuster
## About
Twitter(現X)でAIによる自動返信を行っているBotアカウントを自動的に処理（ブロック/ミュート）するユーザースクリプトです。  
対AI返信海外インプレゾンビに特化しており、認証済みアカウントのみを対象とします。

## Settings
一般条件として、以下の条件に全て一致した場合にBotと判定されます。  
AND条件であるため、**有効条件を減らす**ほどカバー範囲が増えます。
（プロフィール = 表示名 + Bio）

- **1: 表示名に日本語が含まれていない**  
- **2: Bioに日本語が含まれていない**  
- **3: プロフィールにNGワードが含まれている**  
- **4: そのユーザーの返信に日本語が含まれている**  

また、特殊条件として、  
- **A: Bioが空欄である**

があります。当てはまる場合は一般条件を無視して処理を行います。  
誤射の可能性があるため、デフォルトで無効になっています。

その他の設定として、
- **ACTION: 処理の内容、'block'または'mute'** 初期設定はブロック
- **KEYWORDS: プロフに含まれるNGワード** Web3, Crypto, $(キャッシュタグ)など
- **EXCLUDE_FOLLOWED: フォローしているユーザーを除外** 初期設定はtrue
- **EXCLUDE_HIGH_FOLLOWERS: フォロワー数による除外を有効化** 初期設定はtrue
- **EXCLUDE_HIGH_FOLLOWERS_MIN: 指定フォロワー数以上の場合は除外** 初期設定は1万人
- **EXCLUDE_KOREAN: プロフにハングルが含まれている場合は除外** 文化圏が被ることがあるため

があります。

### Recommended Setting（設定例）
- 誤射を確実に無くす場合: 標準設定のまま  
  日本語で返信しているのにプロフに一切日本語がないのは99%Botです。その他キーワード判定等も行います。
- 誤射を避けつつ多めに減らしたい場合: 4を無効、Aを有効化  
  条件4では絵文字や画像のみの返信を含めることが出来ないため無効化。AはBioが空欄の認証ユーザーは大抵Botか怪しいため。
- Web3も暗号通貨も興味ない場合: 3のみ有効化  
  日本語があってもNGワードを使っている場合問答無用で処理します。  
- 日本人しか要らない場合: 1,2,Aのみ有効化  
  プロフに日本語がない認証アカウントを全て処理します。

## Process
日本語判定に関しては、  
ひらがなorカタカナが含まれる場合はtrue、頻出簡体字が含まれる場合はfalse(それ以外の漢字はtrueとする)、その他はfalse  
というロジックになっています。


## Install
Tampermonkeyなどを通じてインストールしてください。
