# FullAutoZombieBuster
## About
Twitter(現X)でAIによる自動返信を行っているBotアカウントを自動的に処理（ブロック/ミュート）するユーザースクリプトです。 
対AI返信海外インプレゾンビに特化しており、認証済みアカウントのみを対象とします。

一般条件として、以下の条件に全て一致した場合にBotと判定されます。

- **1: 表示名に日本語が含まれていない**
- **2: Bioに日本語が含まれていない**
- **3: 表示名かBioに特定のキーワードが含まれている**

また、特殊条件として、  
- **A: Bioが空欄である**

があります。特殊条件は、当てはまる場合は一般条件を無視して処理を行います。  
誤爆の可能性があるため、デフォルトで無効になっています。

## Install
Tampermonkeyなどを通じてインストールしてください。
