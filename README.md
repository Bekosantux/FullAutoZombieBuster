# FullAutoZombieBuster
## About
Twitter(現X)でAIによる自動返信を行っているBotアカウントを自動的にブロック/ミュートするユーザースクリプトです。  
以下の条件に全て一致した場合にBotと判定されます。

1. 表示名に日本語が含まれていない
2. 認証済みアカウントである
3. 表示名/プロフィールに特定のキーワードが含まれている
4. プロフィールに日本語が含まれていない

また、特殊条件として  
- 認証済みでプロフィールに日本語が含まれない場合、他条件によらずブロック

があります。誤爆の可能性があるため、デフォルトでは無効になっています。

## Install
Tampermonkeyなどを通じてインストールしてください。
